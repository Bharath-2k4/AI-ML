# 🎮 Explainable AI for Cheat & Churn Detection in Online Games

An AI-powered web application that detects cheating behavior and predicts player churn in online multiplayer games using Machine Learning, Deep Learning, and Explainable AI (XAI) techniques.

---

## 📖 Overview

This project presents an explainability-focused computational pipeline for detecting game cheats and predicting player churn across multiple analytical views. It combines machine learning models, deep learning, and explainable AI techniques to provide accurate predictions along with understandable explanations.

---

## ✨ Features

- 🔍 Cheat Detection using Machine Learning
- 📉 Player Churn Prediction
- 🧠 Explainable AI using SHAP
- 🖼️ Grad-CAM Visualization for CNN Models
- 👤 Portrait View Analysis
- 📜 Behavior Sequence Analysis
- 🌐 Social Network Analysis
- 🖥️ Interactive Flask Web Application
- 📊 Prediction Confidence Scores
- 📈 Graphical Reports & Visualizations

## 📊 Dataset Views

### 👤 Portrait View
Predicts cheating behavior and player churn using player profile and gameplay statistics.

### 📜 Behavior View
Analyzes player activity logs and gameplay sequences to predict churn.

### 🌐 Social View
Uses player interaction networks to analyze social influence and suspicious activities.

### 🖼️ Image View
Detects visual cheats using CNN and highlights important image regions using Grad-CAM.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Bharath-2k4/AI-ML.git
```

Move into the project directory

```bash
cd AI-ML
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Flask application

```bash
python Backend/app.py
```

Open your browser

```
http://127.0.0.1:5000
```
